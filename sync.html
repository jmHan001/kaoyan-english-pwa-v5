加密云同步

两台设备填写相同的服务地址、公开密钥、同步码和同步密码。密码只保存在当前设备，不会上传。

首次配置 Supabase（约 3 分钟）
  1. 创建一个免费 Supabase 项目，打开 SQL Editor。
  2. 复制项目内 supabase-sync.sql 的全部内容并运行。
  3. 从 Project Settings 的 API 页面复制 Project URL 和 anon public key。
  4. 电脑先生成同步码并保存；手机填写完全相同的四项信息。

同步使用随机同步码定位数据，并用你的密码执行 AES-GCM 端到端加密。请妥善保存同步码和密码，丢失后无法解密云端内容。
