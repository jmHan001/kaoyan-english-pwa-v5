# 拾词 v6.0 本轮修复报告

日期：2026-07-22

## 已完成

- 新词按基础核心、英二核心、共有词、高考基础、考研拓展分层抽取。
- 首次四选一只记录“能认”，不再直接累计长期掌握。
- 已学词再次出现时先自行回忆，再由用户决定是否显示选项。
- 单词小测改为无选项中文核心义输入；旧选择题小测不再用于今日验收。
- 今日小测只从今日学习池里已经背过的词抽取，不混入到期复习或历史错词。
- 两个不同日期的主动回忆加一次语境验证后才进入“已掌握”。
- 答错、点击不会或查看答案会清空当前正确周期并进入重新巩固，历史成功记录保留。
- 复习间隔会根据题型、错误、难度、实际间隔和记忆稳定性调整，不再固定停在 15 天。
- 干扰项优先匹配词性和释义长度，并从全部已学词中取样；近期干扰项会避重复。
- 背词、小测共用独立学习页、判定逻辑与即时系统语音；人工发音仍可通过英音/美音按钮播放。
- 今日学习池、完成标记、收藏、错词、斩词和设置补充冲突合并保护。
- 离线继续本机保存，恢复联网后立即尝试同步，并显示离线、同步中、成功和失败状态。
- 清理原始词库：高考 6008 条归并为 3743 个唯一词条，考研 9602 条归并为 5046 个唯一词条；自动结构扫描为 0 个问题。
- 三种主题保留用户选择；更新版本不再强制切回珍珠白。
- 所有页面恢复浏览器正常缩放；390px 移动端核心页面无横向溢出。

- 高考、考研词库结构审计全部通过。
- 首页、背词、词库、长难句、查词、同步页面在本地静态服务器加载无控制台错误。

- 自动检查只能发现结构问题。词义语义修订继续按实际学习中发现的问题逐条记录到 `VOCABULARY_ISSUES.md`，不使用程序臆测替换。
