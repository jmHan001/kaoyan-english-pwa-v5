# 拾词：APP更新待办与个人记忆规划

## 一、APP更新待办事项

### 1. 本轮总目标

不继续堆叠功能，集中解决三个问题：

1. 单词内容是否准确；
2. 学习结果是否真实；
3. 手机端使用是否顺畅。

### 2. 本轮冻结内容

以下内容暂不修改或扩展：

- 真题模块
- 阅读模块
- 23点学习验收
- ChatGPT / MCP自动导入长难句
- 新主题、新页面和其他新模块

### 3. 单词内容与排序

- [ ] 每日新词优先从基础核心词、英二核心词中抽取
- [ ] 后续再进入高考基础词、考研拓展词
- [ ] 核心程度逐步改为考研词频、阅读重要性与个人错误率共同决定
- [ ] 排查中文翻译、词性、熟词僻义和考研常考义
- [ ] 清理重复词、乱码词、残缺词和格式异常词
- [ ] 修正不相关的近义词和关联词
- [ ] 近义词与关联词同时显示中文核心意思
- [ ] 为词汇修改建立问题清单，避免以后更新重新覆盖

### 4. 解决“假记忆”

当前四选一容易训练熟悉感和排除能力，不能继续把选择正确直接等同于掌握。

学习流程调整为：

1. 初次学习：允许四选一，只标记“识别通过”；
2. 当天回炉：先显示单词并留出回忆时间，再允许显示选项；
3. 跨日验收：隐藏选项，输入一个中文核心义；
4. 语境验证：在例句中判断单词的具体含义；
5. 不同日期完成两次主动回忆后，才标记长期掌握。

具体待办：

- [ ] 四个释义选项保持同词性、相近语义和相近长度
- [ ] 错误选项从全部已学词中抽取
- [ ] 避免连续多题重复同一组错误选项
- [ ] 选择题答对只增加“识别记录”，不能直接增加长期掌握
- [ ] 增加“显示选项”前的无提示回忆阶段
- [ ] 跨日复习使用中文核心义输入或其他客观主动回忆
- [ ] 点击显示答案、不会或回答错误均视为未成功回忆
- [ ] 小测与正式背词采用同一套判定标准

### 5. 重做掌握状态

建议状态改为：

```text
初见 → 能认 → 能想起 → 会运用 → 已掌握
```

- 初见：尚未完成第一次学习
- 能认：四选一能够识别正确释义
- 能想起：没有选项时能回忆核心意思
- 会运用：能够在语境中判断正确含义
- 已掌握：在不同日期的间隔复习中两次主动回忆成功

待办：

- [ ] “连续正确3次”只保留为本轮学习指标
- [ ] 同一天重复答对不能直接进入已掌握
- [ ] 答错后清零本轮连续正确次数
- [ ] 答错不删除全部历史学习成果，只降低记忆稳定性
- [ ] 统计页区分识别通过、本轮通过、待验证和长期掌握
- [ ] 斩词独立统计，不能自动算作已掌握

### 6. 调整复习算法

当前固定间隔为：

```text
10分钟 → 1天 → 3天 → 7天 → 15天
```

问题是简单词与困难词采用相同间隔，并且最后长期停留在15天，后期复习量会持续增加。

改进方向：

- [ ] 根据单词难度、历史错误、成功回忆次数和实际间隔计算下次复习
- [ ] 简单词快速延长至30天、60天和更长时间
- [ ] 困难词维持较短间隔并提高回炉频率
- [ ] 答错后当天回炉，并缩短下一次跨日复习
- [ ] 延迟较久仍能答对时，明显延长下一次间隔
- [ ] 到期复习继续与每日新词分开统计
- [ ] 根据当天复习压力建议新词数量，但不能自动修改用户学习池

### 7. 背词、语音与英文交互

- [ ] 进入单词时自动朗读
- [ ] 每次答对后立即朗读正确单词
- [ ] 答错后朗读正确答案对应的单词
- [ ] 单词小测使用与背词页一致的语音逻辑
- [ ] 语音加载失败不能阻塞答题或下一词
- [ ] 防止快速点击造成语音重叠
- [ ] 可交互英文单词支持点击朗读和查看词义
- [ ] 例句支持完整朗读
- [ ] 词根、例句、搭配放在作答后展示，避免提前泄露答案

### 8. 学习池与任务记录

- [ ] 今日学习池生成后保持固定，刷新不能换词
- [ ] 只有用户主动操作才能新增、替换或补满单词
- [ ] 自定义每日数量后，主页任务总数同步变化
- [ ] 加背后任务总数与学习池同步变化
- [ ] 今日任务完成后，刷新、退出和重新打开仍然保持完成
- [ ] 联网同步后不能把完成状态退回未完成
- [ ] 右滑返回上一词只作回看，不重复计分
- [ ] 今日完成条件和剩余数量在主页清晰显示

### 9. 错词、收藏、复习与斩词

- [ ] 错词回炉包含全部历史错词
- [ ] 用户可以选择本次随机重温数量
- [ ] 错词再次答错后清零本轮连续正确
- [ ] 收藏图标在背词卡片右上角正常显示
- [ ] 重点词表在手机端排版正常
- [ ] 收藏和取消收藏后立即更新
- [ ] 已背单词按时间和状态正确显示
- [ ] 斩词可以恢复，恢复后原学习记录仍存在

### 10. 手机端用户体验

- [ ] 今日背词从主页进入独立大页面
- [ ] 背完后提供明确的返回主页入口
- [ ] 删除重复背词入口和拥挤的小窗口
- [ ] 检查不同主题的文字和按钮可见性
- [ ] 修复按钮文字缺失、横线贴字、卡片溢出和收藏排版
- [ ] 重要按钮有点击反馈和加载状态
- [ ] 空学习池、无错词和无收藏状态提供明确说明
- [ ] 检查iPhone安全区、底部导航和横向溢出
- [ ] 保留正常可访问性缩放，避免误触双击放大

### 11. 同步的正确边界

无法在没有网络时进行真正同步。

正确流程是：

```text
离线学习 → 本机保存 → 恢复联网 → 立即尝试补同步
```

待办：

- [ ] 离线时不影响学习和本地记录
- [ ] 恢复联网后立即触发同步尝试
- [ ] 明确显示等待联网、同步中、成功和失败
- [ ] 同步失败不能清空或覆盖本地数据
- [ ] 手机和电脑先后修改后能够正确合并
- [ ] 同步后核对学习池、错词、收藏、斩词、长难句和设置

### 12. APP最终验收标准

- [ ] 不增加范围外功能
- [ ] 单词及关联内容无明显错误
- [ ] 不能只靠排除选项获得“已掌握”
- [ ] 长期掌握经过跨日主动回忆验证
- [ ] 今日任务刷新、重启和同步后不丢失
- [ ] 背词、小测、错词、收藏和复习完整可用
- [ ] 手机端入口明确、显示清晰、操作顺畅
- [ ] 离线正常保存，恢复联网后自动补同步

## 二、建议记忆方法规划

### 1. 先明确当前问题

目前容易忘，不一定代表记忆能力差，主要可能来自：

- 长期通过选择题形成熟悉感，而不是真正回忆
- 一次学习太多释义和附加信息
- 同一天重复很多次，产生短期记住的错觉
- 每日新词量超过第二天能够保持的数量
- 睡眠、压力和注意力影响记忆巩固

判断是否记住的标准不是“刚背完会不会”，而是“隔天没有选项还能不能想起来”。

### 2. 单词学习原则

#### 原则一：先回忆，再看答案

看到英文单词后，先给自己3秒，说出一个中文核心意思。

- 能说出：再检查答案
- 说不出：按不会处理
- 靠排除选对：只算见过，不算记住

#### 原则二：一个词先记一个核心义

第一次不要同时背完所有释义、词组和熟词僻义。

顺序应为：

```text
核心义 → 常见搭配 → 考研常考义 → 熟词僻义
```

#### 原则三：跨日回忆比当天重复重要

建议复习节点：

```text
当天 → 第2天 → 第3天 → 第7天 → 第15天 → 第30天
```

具体间隔以后由APP根据答题表现自动调整。

#### 原则四：答错要真实记录

- 3秒内完全想不起来就算不会
- 不用因为答错而焦虑
- 答错提供了最有价值的复习信息
- 不要为了维持正确率依赖选项和排除法

### 3. 每日执行方案

如果当前每天30个新词遗忘明显，先暂时降低到10至15个，等隔天保持率提高后再增加。

建议每天分为三段：

#### 早上：到期复习，约10分钟

- 先完成旧词复习
- 尽量无提示回忆
- 不会的词诚实标错

#### 白天：学习新词，约15分钟

- 每天10至15个起步
- 先学习核心词
- 每个词先记核心义和发音
- 作答后再看例句、搭配和词根

#### 晚上：快速验收，约5分钟

- 只看英文说中文核心义
- 3秒说不出就算不会
- 不重新完整背一遍
- 把时间留给真正想不起来的词

### 4. 每周评价方式

不要用当天正确率判断记忆能力。

每周只观察：

- 隔天无提示回忆率
- 7天后仍能想起的比例
- 依靠选项答对的比例是否下降
- 错词数量是否逐渐稳定
- 每日任务是否可以持续完成

建议目标：先让隔天主动回忆率稳定达到70%左右，再逐步增加每日新词。

### 5. 生活与注意力

- 尽量保持固定睡眠时间，每晚保证约7小时或以上睡眠
- 避免熬夜后继续增加新词量
- 学习期间关闭短视频和频繁通知
- 连续学习20至25分钟后短暂休息
- 保持日常活动或散步
- 压力明显增大时先减少学习量，优先保持连续性
- 不依赖所谓增强记忆的保健品

### 6. 需要留意的情况

如果遗忘不仅发生在背单词，还经常忘记刚发生的谈话、约定、路线或重要日常事项，并且持续加重或影响正常生活，应及时向医生说明情况，不要自行判断原因。

## 最终原则

```text
选择题用于帮助学习
主动回忆用于证明掌握
少背一点，隔天记住
```
